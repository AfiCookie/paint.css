<h2>Install</h2>
<p><strong>From CDN:</strong> Add the following link to the end of your <code>&lt;head&gt;</code> section.</p>

```html
<link rel="stylesheet" href="paintcss.cerium.xyz" crossorigin="anonymous" />
```
